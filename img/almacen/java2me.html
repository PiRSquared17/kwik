El Nokia 3100 es un teléfono de la Serie 40 1.0 de Nokia.
Resolución: 128x128x4096 (12 bits por color)
Especificación JVM: MIDP 1.0 con extensiones Nokia UI API


Todos los teléfonos que soportan Java son MIDP 1.0. Es un estándar un poco limitado a la hora de hacer juegos con buenos gráficos: no soporta doble buffer; trabajo con sprites, colisiones y scroll corre de cuenta del programador; y lo peor, no funciona a pantalla completa. El código que escribas para teléfonos con esta especificación funcionará sin problemas en cualquiera de ellos, con las limitaciones gráficas de cada uno (resolución, colores...).
Los fabricantes amplían este estándar con funciones adicionales, no compatibles entre teléfonos de distinto fabricante. Por ejemplo Nokia tiene sus funciones UI (User Interface) que añaden pantalla completa, reflexiones en sprites... Sin embargo, como ya he dicho la pega es que solo funciona en los Nokia.
Finalmente, los teléfonos modernos soportan MIDP 2.0. Programar en este estándar es muy cómodo por la cantidad de métodos que incluye de cara a juegos: sprites, scroll, pantalla completa, sonidos... De este modo nuestra aplicación funcionará en todos los teléfonos de este estándar.


Para programar aplicaciones J2ME es preciso bajarse:
- el Java 2 SE Software Developmeent Kit (SDK) 1.4. 51,5 Mb

Si quieremos desarrollar para MIDP 1.0 o 2.0 necesitamos: - el Java 2 ME Wireless Toolkit ((WTK) 2.2. 16,9 Mb

Todo está disponible gratis en la web de Sun: http://java.sun.com

Por el contrario, si queremos trabajar sólo para los teléfonos Nokia, hay que conseguir:
- el Nokia Developer's Suite for J2ME 2.2. 47,4 Mb
- el SDK correspondiente a tu teléfono, een mi caso el Nokia 7210 SDK 1.0. 18,2 Mb

Todo está disponible gratis en la web de Nokia: http://forum.nokia.com

Aquí un servidor se lo ha bajado con su módem 56k. :D

El juego es compatible con todos los Nokia con pantalla a color, con una resolución mínima de 128x128. Para descargártelo debes entrar con tu móvil en http://www.geocities.com/dan_yomismo/index.wml
Esto se hace en el menú Servicios -> Ir a dirección
El juego se almacena en la carpeta de Aplicaciones. Se almacenaría en la de Juegos si los señores de geocities posibilitasen subir archivos .jad :/
